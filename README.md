
## Implementation details and technologies

### Project based on 3-layer architecture:
>- Presentation layer (controllers)
>- Application layer (services)
>- Data access layer (Repositories)

### Technologies
>- Spring Boot
>- MySQL
>- Lombok
>- Maven
